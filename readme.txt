# Evaluation d'entraînement n°2 - Studi 

## Contexte
Nous allons réaliser, dans le cadre, de l'évaluation study un site vitrine pour une association de joueur de baskeball fictive.
Cette association se situerai dans le 12e arrondissement de Paris. Le but est de leur créer un site internet permettant d'augmenter leur visibilité.

## Objectif du projet
- Structure sémantique html et utilisation des méta pour optimiser le référencement
- Intégration mobile first

## Projet
Afin de réussir ce projet, j'ai utiliser de l'html et du css ainsi que le framework Bootstrap.
Mon objectif personnelle était d'utiliser un maximum les classes de Bootstrap afin d'avoir une page de style petite.
Je me suis aussi concentré sur le responsive et sur le référencement.
